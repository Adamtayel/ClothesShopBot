## Clothes Shop Telegram Bot 

A simple Telegram bot to browse and display products from a CSV catalog.

### Features
- Display products by type
- Show product details and images
- Fully interactive using inline keyboard

### Deployment
- Hosted on [Railway](https://railway.app/) for 24/7 uptime
- Environment variables used for Telegram bot token security

### Tech Stack
- Python 3
- python-telegram-bot library
- CSV for product data
- Railway for cloud deployment
